# 💰 MoneyManager: Personal Finance App
**Vibecoded with Precision | Product-First Development**

MoneyManager is a modern personal finance tracking application designed for Android and Mac browsers. This project serves as a showcase of high-velocity "Vibecoding"—leveraging AI as a pair programmer to move from PRD to a functional MVP in record time.

## 🚀 The Vision
As a Product Manager, I built this to solve the friction of manual expense logging. The goal was to create a "flat and fast" UI that mimics the efficiency of ERP systems I manage professionally, but for personal use.

## ✨ Key Features
* **Intuitive Dashboard:** Real-time visualization of spending vs. budget.
* **Cross-Platform Design:** Unified experience across Android and Mac browsers.
* **ERP-Inspired Logic:** Built with data integrity and clean categorization at the core.
* **Security First:** Local-first data handling with AppleCare+ style reliability.

## 🛠 Tech Stack (The Vibecoding Workflow)
* **Core:** Android SDK, Kotlin/Java, Python (Backend)
* **AI Augmented:** Built using Google Gemini for rapid prototyping and debugging.
* **Version Control:** Git/GitHub for CI/CD readiness.
* **Documentation:** Driven by a comprehensive PRD (Product Requirement Document).
